# StakeMyGoal v4 - Full MVP Project

This project is a frontend MVP built with Next.js 13 (TSX ready). It uses localStorage as a mock backend and includes a simulated Stripe payment flow.

Pages included:
- / (landing)
- /create-goal
- /dashboard
- /payment (simulated)
- /transactions
- /profile
- /terms
- /privacy

To run:
1. npm install
2. npm run dev
3. Open http://localhost:3000

# StakeMyGoal v2 - StickK-inspired MVP

This version includes:
- Improved landing page and hero
- Create Goal page with deadline, stake, fee calc (placeholder payment)
- Dashboard with mock data (stored in localStorage), editable goals, complete/forfeit actions
- Dark/light mode toggle (persisted in localStorage)
- Placeholder Stripe API route

How to run:
1. npm install
2. npm run dev
3. Open http://localhost:3000 or run in StackBlitz / Codespaces (Next.js 13)

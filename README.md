# StakeMyGoal v5 - Neon + Drizzle Integrated MVP

This build adds Neon (Postgres) + Drizzle ORM integration.

Important:
1. Copy .env.example to .env.local and set DATABASE_URL to your Neon connection string.
2. Install dependencies: npm install
3. Run dev: npm run dev
4. The API routes /api/goals and /api/payments use Drizzle and require DATABASE_URL to be set.

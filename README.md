# StakeMyGoal v3 - Enhanced MVP

What's new in v3:
- Create goal page includes deadline and validator fields.
- Clear UI message that validator lowers fee (£2 vs £3.50).
- Dummy payment page to simulate Stripe transaction; transactions recorded to localStorage.
- Dashboard shows deadlines, validator, motivational elements (streak, tips, celebrate).
- You can add goals with deadline and validator from quick-add, and edit goals including deadline.
- Dark/light mode remains.

How to run locally or on StackBlitz:
1. npm install
2. npm run dev
3. Open http://localhost:3000
